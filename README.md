# Tropospheric layer cloud worker
---------------------------------
Data-mining Cloud Worker

### USE
- Install requirements
```
npm install
```
- start default global memory server
```
npm run global
```
- Start local memory server
```
npm run local
```
- test connection
```
npm run client
```
- deploy
```
cf login
cf push
```
